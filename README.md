
# 🌎 Around The U.S.

Projeto desenvolvido como prática de JavaScript, manipulação de DOM e validação de formulários.

## 📌 Sobre o Projeto

"Around The U.S." é uma aplicação web interativa onde o usuário pode:

- ✏️ Editar informações do perfil
- ➕ Adicionar novos cartões
- ❤️ Curtir cartões
- 🗑 Excluir cartões com pop-up de confirmação
- 🔍 Visualizar imagem ampliada em modal
- ❌ Fechar modais ao clicar fora ou pressionar ESC
- ✅ Validação de formulários em tempo real

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Manipulação de DOM
- Validação nativa de formulários
- JavaScript Assíncrono com APIs

---

## 🧠 Funcionalidades Implementadas

### 🔹 Edição de Perfil
- Preenchimento automático com dados atuais
- Validação de campos obrigatórios
- Botão desabilitado até formulário válido

### 🔹 Criação de Cards Dinâmicos
- Uso de `<template>`
- Renderização dinâmica com `cloneNode`
- Eventos adicionados via JavaScript

### 🔹 Interações nos Cards
- Toggle de like
- Remoção do card
- Abertura de modal com imagem

### 🔹 Sistema de Modais
- Funções reutilizáveis:
  - `openModal()`
  - `closeModal()`
- Fechamento por:
  - Botão de fechar
  - Clique fora do conteúdo
  - Tecla `Escape`

### 🔹 Validação
- Exibição de mensagens de erro personalizadas
- Estado do botão controlado dinamicamente
- Manipulação de classes para feedback visual
